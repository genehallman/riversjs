# Rivers

## To contribute:

* Before starting, make sure you have mongo installed and running (not explained here - google it).
* `git clone git@github.com:genehallman/rivers.git`
* `cd rivers`
* `npm link`
* `cd ..`
* `rivers generate foo`
* `cd foo`
* `npm link rivers`
* `rivers serve`
